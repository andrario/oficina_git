Configurando o git (para usuários de linux)

  • Faça o download do arquivo: gcm-linux_amd64.2.0.785.deb
  • Acesse o diretório do arquivo baixado através do terminal.
  • Execute o seguinte comando: sudo dpkg -i nome_do_arquivo.deb
  • Então execute o seguinte comando: git-credential-manager-core configure
  • Por fim execute o comando: git config --global credential.credentialStore cache
